粒子群算法

介绍

粒子群优化（Particle Swarm Optimization, PSO），又称粒子群算法、微粒群算法。PSO 算法最初是为了图形化地模拟鸟群优美而不可预测的运动。而通过对动物社会行为的观察，发现在群体中对信息的社会共享提供一个演化的优势，并以此作为开发算法的基础[1]。通过加入近邻的速度匹配、并考虑了多维搜索和根据距离的加速，形成了 PSO 的最初版本。之后引入了惯性权重w来更好的控制开发（exploitation）和探索（exploration），形成了标准版本。

PSO 算法属于一种万能启发式算法，能够在没有得知太多问题信息的情况下，有效的搜索具有庞大解空间的问题并找到候选解，但同时不保证其找到的最佳解为真实的最佳解

算法原理

PSO 算法是基于群体的，根据对环境的适应度将群体中的个体移动到好的区域。然而它不对个体使用演化算子，而是将每个个体看作是 DDD 维搜索空间中的一个没有体积的微粒（点），在搜索空间中以一定的速度飞行，这个速度根据它本身的飞行经验和同伴的飞行经验来动态调整。第 iii 个微粒表示为 Xi=(xi1,xi2,…,xiD)X_i =(x_{i1}, x_{i2}, …, x_{iD})Xi​=(xi1​,xi2​,…,xiD​) ，它经历过的最好位置（有最好的适应值）记为 Pi=(pi1,pi2,…,piD)P_i = (p_{i1}, p_{i2}, …, p_{iD})Pi​=(pi1​,pi2​,…,piD​)，也称为 pbestpbestpbest。在群体所有微粒经历过的最好位置的索引号用符号 ggg 表示，即 pgp_gpg​，也称为 gbestgbestgbest 。微粒 iii 的速度用 Vi=(vi1,vi2,…,viD)V_i = (v_{i1}, v_{i2}, …, v_{iD})Vi​=(vi1​,vi2​,…,viD​) 表示。对每一代，它的第 d+1d+1d+1 维(1≤d+1≤D)(1 \leq d+1 \leq D)(1≤d+1≤D)根据如下方程进行变化：

算法步骤

标准 PSO 的算法流程如下：

  • 初始化一群微粒（群体规模为 mmm ），包括随机的位置和速度；
  • 评价每个微粒的适应度；
  • 对每个微粒，将它的适应值和它经历过的最好位置 pbestpbestpbest 的作比较，如果较好，则将其作为当前的最好位置pbestpbestpbest；
  • 对每个微粒，将它的适应值和全局所经历最好位置 gbestgbestgbest 的作比较，如果较好，则重新设置gbestgbestgbest的索引号；
  • 根据方程（1）变化微粒的速度和位置；
  • 如未达到结束条件（通常为足够好的适应值或达到一个预设最大代数 GmaxG_{max}Gmax​），回到（2）。

算法参数

PSO 参数包括：

  • 群体规模 mmm

  • 惯性权重 www: 惯性权重www使微粒保持运动的惯性，使其有扩展搜索空间的趋势，有能力探索新的区域。

    • 如果没有第一部分，即 w=0w = 0w=0，则速度只取决于微粒当前的位置和它们历史最好位置 pbestpbestpbest 和 gbestgbestgbest ，速度本身没有记忆性。假设一个微粒位于全局最好位置，它将保持静止。而其它微粒则飞向它本身最好位置 pbestpbestpbest 和全局最好位置 gbestgbestgbest 的加权中心。在这种条件下，微粒群将统计的收缩到当前的全局最好位置，更像一个局部算法。

    • 在加上第一部分后，微粒有扩展搜索空间的趋势，即第一部分有全局搜索的能力。这也使得www的作用为针对不同的搜索问题，调整算法全局和局部搜索能力的平衡。

  • 加速常数 c1c_1c1​ 和 c2c_2c2​: 加速常数 c1c_1c1​ 和 c2c_2c2​ 代表将每个微粒推向 pbestpbestpbest 和 gbestgbestgbest 位置的统计加速项的权重。低的值允许微粒在被拉回来之前可以在目标区域外徘徊，而高的值导致微粒突然的冲向或者越过目标区域。

    • 如果没有后两部分，即 c1=c2=0c_1 = c_2 = 0c1​=c2​=0，微粒将一直以当前的速度飞行，直到到达边界。由于它只能搜索有限的区域，将很难找到好的解。
    • 如果没有第二部分，即 c1=0c_1 = 0c1​=0，则微粒没有认知能力，也就是“只有社会（social-only）”的模型。在微粒的相互作用下，有能力到达新的搜索空间。它的收敛速度比标准版本更快，但是对复杂问题，比标准版本更容易陷入局部优值点。
    • 如果没有第三部分，即 c2=0c_2 = 0c2​=0，则微粒之间没有社会信息共享，也就是“只有认知（cognition-only）”的模型。因为个体间没有交互，一个规模为mmm的群体等价于mmm个单个微粒的运行。因而得到解的几率非常小。

  • 最大速度 VmaxV_{max}Vmax​，VmaxV_{max}Vmax​ 决定在当前位置与最好位置之间的区域的分辨率（或精度）。如果 VmaxV_{max}Vmax​ 太高，微粒可能会飞过好解，如果 VmaxV_{max}Vmax​ 太小，微粒不能进行足够的探索，导致陷入局部优值。该限制有三个目的：

    • 防止计算溢出；
    • 实现人工学习和态度转变；
    • 决定问题空间搜索的粒度

  • 最大代数 GmaxG_{max}Gmax​。
